# 용어정리
* 서블릿 : Controller, HttpRequest, HttpResponse를 추상화해 인터페이스로 정의해 놓은 표준.
* 서블릿 컨테이너 : 서블릿 컨테이너는 이 서블릿 표준에 대한 구협을 담당. (웹 서버)
* Embedded 톰캣 : jar 파일을 이용해서 톰캣서버를 실행할 수 있다.
* 서블릿의 한계를 (동적 html)를 극복하기 위해 등장한 것이 jsp
